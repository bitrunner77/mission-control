# 🚀 Mission Control - Quick Start Guide

Get up and running with Mission Control in under 5 minutes.

---

## ⚡ Fastest Way to Start

1. **Open the file**
   ```bash
   # macOS
   open mission-control/index.html
   
   # Linux
   xdg-open mission-control/index.html
   
   # Windows
   start mission-control/index.html
   ```

2. **That's it!** The onboarding tour will start automatically.

---

## 🎯 First Steps

### 1. Complete the Onboarding Tour (2 min)
- Follow the 6-step interactive tour
- Learn key features and shortcuts
- Can be restarted anytime from Shortcuts tab

### 2. Explore the Dashboard (1 min)
- See your active projects
- Check priority tasks
- Review revenue metrics

### 3. Try the Kanban Board (1 min)
- Drag a task between columns
- Click "Add Task" to create new cards
- Use arrow keys for keyboard navigation

### 4. Check AI Planning (1 min)
- See AI-recommended next actions
- Review bottleneck analysis
- Get daily focus suggestions

## ⌨️ Essential Shortcuts

| Shortcut | What It Does |

## 🆘 Troubleshooting

### "Page looks broken"
- Make sure you're opening `index.html` directly
- Check that Tailwind CSS CDN is loading (requires internet)

### "Data doesn't save"
- Enable localStorage in your browser
- Don't use private/incognito mode

### "Keyboard shortcuts don't work"
- Click on the page first (not in an input)
- Check if Caps Lock is on
